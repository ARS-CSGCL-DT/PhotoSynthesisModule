Photosynthesis Module Overview

Introduction

This is a coupled model of leaf gas-exchange and photosynthesis-stomatal conductance-energy balance for a maize leaf. The C++ GasExchange class encapsulates all the calculations to estimate CO2 assimilation rate, stomatal conductance, leaf temperature, and transpiration for a square meter of leaf. The calculations are based on von Caemmerer(2000) C4 model, Kim and Leith (2003) C3 rose model, BWB stomatal conductance(Ball et al., 1987) and energy balance model as described in Campbell and Norman (1998)
