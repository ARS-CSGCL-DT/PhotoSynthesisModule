Photosynthesis Module

Introduction

This is a C++ module to simulate gas exchange processes in a leaf. The Gas Exchange class encapsulates all the calculations to estimate photsynthetic rate, CO2 assimilation, stomatal conductance, and transpiration for a square meter of leaf.

This is a coupled model of photosynthesis-stomatal conductance-energy balance for a maize leaf
this unit simulates Maize leaf gas-exchange characteristics including photosynthesis,
traspiration, boundary and stomatal conductances, and leaf temperature based
on von Caemmerer (2000) C4 model, BWB stomatal conductance (1987) and
Energy balance model as described in Campbell and Norman (1998)
